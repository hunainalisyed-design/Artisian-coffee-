# Artisian-coffee-
coffee shop 
A modern coffee house landing page website built with HTML and CSS.
This project focuses on creating a premium café-style user interface with warm colors, elegant typography, and smooth layout design.

The goal of this project is to practice frontend layout design, styling techniques, and aesthetic UI development while building a realistic coffee brand website.

🌐 Live Preview

You can open the project locally or host it using GitHub Pages.

✨ Features

Elegant coffee shop themed UI

Warm dark aesthetic design

Smooth scroll behavior

Custom CSS variables for theme colors

Modern typography using Google Fonts

Decorative background effects

Responsive layout structure

Clean and organized code

🛠️ Built With

HTML5 – Structure of the website

CSS3 – Styling and layout

Google Fonts – Typography

Custom CSS Animations & Effects
